# Visual-Algorithms
This repository contains an interactive web application that visualizes various sorting algorithms. The project, built with React and styled using the Carbon Design System, provides a clear and intuitive way to understand how different sorting methods work.
<br>
<br>

# Features
Interactive Visualizations: Watch algorithms like Merge Sort, Quick Sort, and Bubble Sort rearrange data in real-time.

Algorithm Comparison: Easily switch between different sorting algorithms to compare their performance and efficiency.

Customizable Inputs: Adjust the size and type of the data array to see how algorithms handle different scenarios.

Step-by-Step Breakdown: A step-by-step feature will be included to allow users to pause and analyze the process for better learning.

<br>
<br>

# Technologies Used
React: For building the user interface.

Carbon Design System: For a consistent and accessible UI.

JavaScript: For implementing the core logic of the sorting algorithms.

<br>
<br>

# Getting Started
Clone the repository.

Install dependencies with npm install.

Run the application with npm start.
